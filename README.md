# 🏔️ Celtic Trekking - Premium Travel & Adventure Platform

Welcome to the official repository for **Celtic Trekking**, a high-end dynamic reservation and content-driven platform built for travel agencies specializing in international trekking expeditions (Nepal, Tibet, Morocco, Romania).

This project features a fully custom, responsive frontend experience paired with a robust administrative dashboard powered by **Filament v5** and **Laravel 13**.

## ✨ Features

### 🖥️ Frontend Client Experience
* **Dynamic Grid Megamenu:** Widescreen header navigation containing graphical destination preview cards with modern hover transformations.
* **Responsive Layout:** Fluid breakpoints optimized across Mobile, Tablet, and Desktop screens.
* **Dynamic Destinations:** Pages loaded seamlessly via slugs pulled from the database backend.
* **Interactive Contact & Booking:** Customer interaction portals with validation for customized booking queries.

### 🛡️ Administrative Dashboard (Filament v5)
* **Dynamic Site Config System:** Single-view setting manager that intelligently serves a visual `FileUpload` matrix for branding graphics (e.g., Logo) while adapting to `Textarea` editors for scripts, meta tags, and inline iframes.
* **Live Inquiry Sidebar Badges:** Real-time counter metrics highlighting unread customer contact submissions inside the menu tree using dedicated layout loops.
* **Direct Email Response Modals:** In-dashboard `RichEditor` workflow allowing administrators to instantly reply to user inquiries directly via SMTP/API mail integrations.
